Class: Level

Level(clouds, backgrounds, bottles, coins, enemies, endboss)

Represents a game level with various elements such as clouds, backgrounds, bottles, coins, enemies, and an end boss.

Constructor

new Level(clouds, backgrounds, bottles, coins, enemies, endboss)

Initializes a new level instance with given elements.
Parameters:
Name Type Description
clouds Array Clouds present in the level.
backgrounds Array Background layers for the level.
bottles Array Bottles to collect in the level.
coins Array Coins to collect in the level.
enemies Array Enemies encountered in the level.
endboss Object The end boss of the level.
