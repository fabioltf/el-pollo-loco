Class: Backgrounds

Backgrounds(img, axisX)

Represents a background sprite in the game. Inherits from the Sprite class and handles the properties and behavior of a background image.

Constructor

new Backgrounds(img, axisX)

Creates an instance of the Backgrounds class.
Parameters:
Name Type Description
img string The source URL of the background image.
axisX number The x-axis position of the background.
