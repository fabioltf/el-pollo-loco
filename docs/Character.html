Class: Character

Character()

Represents a character in the game, extending the Sprite class. Manages the character's animations, movements, sounds, and state.

Constructor

new Character()

Creates an instance of the Character class. Loads the initial character image and all animations for walking, idling, sleeping, jumping, hurting, and dying. Initializes the character's animation and gravity.
